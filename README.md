Mortgage Form
---------------------------------------------

This project contains technologies:
- Spring Boot 2.1.8
- JPA + Hibernate
- Maven 3.1.1
- Java 11
- Thymeleaf 3
- Bootstrap 4

How to run application
---------------------------------------------
```bash
## Build application from base directory
mvn clean install

## Run initialization sql scripts
initial-data-forms.sql
initial-data-users.sql

## Run Spring boot application
mvn spring-boot:run

```
